创作之路

发表于 2016-08-23   |   分类于 写作、创作   |     |   阅读次数

首先在此介绍搭建站点的原由，当我发现我需要经济独立时，我总在想我手里有什么武器，而当我细细想想，确实没什么，那么我要做什么，又能做什么，常常也就不免陷入什么是自由经济人的思考。着眼眼前，什么变现最快，当然是入驻互联网时代，这也是我建站初衷，当然也想试着写作、创作、分享生活、把学习当成一种生活方式。也许并不弄出个什么名堂，但试着前行总是好的吧，至少算是简单的自我掌控了。

首先谈谈为什么执着于创作

  • 创作对绝大多数人而言是带有恐吓意味，逃避即输
  • 创作是自我意识表达的极有意义的武器
  • 人人都能创作只是表现形式不同
  • 知识产权日益得到重视产权变现更加容易

总结而言，创作的意义在于革新，而更深层次的作用是表达个性，现如今缺什么不可以，我看缺个性最糟，所以我说别老看别人怎么去看，会怎么说，静下来去做，试着锻炼自己的专注【冥想–如何专注于心–就是一种好的方式】比那啥总要好，而且创作绝对是对思维最好的研磨，思维形式一定程度决策了行为模式，行为影响事件发展，而大多时候我们倾向事件往好的方面发展，这一定反驳了宿命论主义吧，可以说这是一个简单道理可是很少人肯实践，我想想赢的人总会走上这条路，崎岖但快乐。

创作的表现形式在于内在鉴赏水准的表达

1.设计模式的喜好

入职前端设计工程师（工程师——相当高大上——算是行行的顶级能够灵活运用工程学）的理由不是技术攻关的趣味，而在于某些迷人的设计理念，而很多时候是可以融入各类产品设计、生活细节的。这里简单介绍几种笔者关注的设计理念模式：

  • 1.1.极简主义——所见即所得,真正意义纯粹

  • 1.2.包豪斯风格——反主流（创建之初）现代主义写实

包豪斯风格起初为表现主义，尤其受到约翰·伊登的个人直觉式、神秘主义的教学影响很大，与格罗佩斯理想中的“集体创作”大相径庭，也偏离了包豪斯与工业生产结合的宗旨，学生在伊登神秘、权威的偶像式教育下，所展现出的自然崇拜与自我表现的神秘主义，已经与当时工业设计训练的理性立场违背。
1921年，在柏林郊区，由木材商人阿道夫·桑玛菲尔德（AdolfSommerfeld）委托包豪斯木工工厂的私人住宅建筑案，是表现主义时期的揭幕作品夏日屋（HausSommerfeld），室内所有设计皆为包豪斯木工部门学生作品。
1921年，荷兰风格派运动（DeStijl）精神领袖凡·杜斯堡（TheovanDoesburg）造访魏玛，对于同样追求知性与秩序、工业发展导向的包豪斯，被伊登的神秘主义风格所垄断感到震惊，并提出严厉批评，1922年杜斯堡在魏玛举办“风格派新艺术演讲”，主要批判表现主义的弊病，以及阐述构成主义的理论，规律富有秩序、非个人的、理性化的设计风格，与包豪斯的工业生产、面向实际的宗旨相吻合。风格派开始影响包豪斯，同时也让格罗佩斯慎重地考虑往后的走向，他开始放弃战后初期的乌托邦理想与手工艺倾向，提出应该从工业化倾向来发展设计教育的理念。
为了挽救包豪斯的方向与误解，1922年6月，格罗佩斯聘请了瓦西里·康定斯基（Wassily Kandinsky），借由他理性的科学理论课程让教学重新走上轨道。1922年10月，格罗佩斯公开劝退伊登辞职，并在1923年由拉士罗·摩荷里·那基（LászlóMoholy-Nagy）取代他的职务，在保罗·克利、康定斯基与那基的教学努力下，包豪斯风格逐渐走向理性主义与构成主义。
1923年，在包豪斯第一次展览会上展出的模型屋非洲号角（MusterhausAmHorn）由乔治·莫奇（GeorgMuche）设计，简单的立方体外观，利用钢筋与混凝土结构建成，每个房间都有明确不可取代的功能，室内灯光由那基设计，家具皆为马歇尔·布鲁耶（Marcel Breuer）的作品，具新即物主义（NeueSachlichkeit）的组成与荷兰风格派运动的特色，有着“冷，极简，机械”的意象。

  • 1.3.蒙太奇——意识流动如何创建连动画面（这倒是让我想起一部电影#两杆大烟枪#）

  • 1.4.工程学设计框架模式——架构师如何搭建思维大厦

  • 1.5.原生态设计理念——传达跟人性的体验（原生态即还原生态是我拟化的一个表意词，而就我小见，Amazon在做这方面，水墨印刷kindle，也有开发水彩版，我想这会是一种趋势）

2.认知水平的广度和高度

我总在想人与人还是有差距的，存在天赋鸿沟，但却又不是不可逾越，只是懂得的人知道去做更多，看更多尝试更多，试着做错事总会对一次。我想我是欠缺那份精神，当我的纠结与懒癌综合症发了时表征为不作为，但我想我会坚持去做到，有一天我也能做到顶级。

  • 2.1.如何到达广泛认知

很多人认为专注做一件事，专注于一个领域，才会做的更好，但我并不认同，因为专注于一件事只是基于一个时间点上的微观层面，这并不意味着不能多线轴并发做事，而其实实践证明效率更高。事务之间总是有内在联系，查理芒格的普世智慧在于花时间尽量研究多个域顶级知识层面（总能融汇些许），当灵感突兀而来，很多时候我们太过着迷于一点，多数时候好的灵感会消亡，原因在于我们看到的点还是太少，我们也许可以偶尔看到成线的两点，当我们试着去看四个点时，我们可能看到四面体，但其实我们看到的可能还是和三个点的面一样，如果最终我们能看到很多点构成了不起的思维大厦时，创作注定会伟大而非凡，所以生活上应去看更广的面，而且生活总得给自己找些新乐子吧！

  • 2.2.有质量的认知

这是我一个固执的偏见的，我总觉的读鸡汤类的书籍的人其实摆脱不了懦弱的本性，这像一个循环式环路，总会回到起点，因为本质上的问题并未解决，而教科书应该是最有营养的但总枯燥单调。而当我看书的时候我会看什么，可选择的其实很多，而且足够高质量，好东西太多而时间又有限，所以知道自己缺什么就要补什么，当我觉得我很缺坚毅性格时[原由那根植内心深处可怕的纠结加懒癌]，所以我会多看性格长成类小说，记忆比较深的有昂利沙里叶《越狱》+《赌注》、麦卡锡的《路》，不试着坚定前行怕永远看清脚下的路也不会认同自己。
对于非虚构类型的书，小众的不如大众的，权威的最新的往往足够全面，当我觉得却什么，我就要补什么所以总会进步吧，但我缺钱时我要怎么做，如何认知，很简单的说当然是理财，那我如何该狩猎高质量理念，简单而言就是多看，不比较是不会知道什么更好，而学习了马上用是最好的检验，强有力的行动力总会带来意想不到的好处。

3.表现形式

谈表现形式在于品质生活，我想活着的人总想好的过活，品质生活在于我们拥有选择生活的细节自由，自由之初在于对内在品趣的外显表征，当我们想有不一样的过活时，我们会更注重表现形式。

  • 对生活的高度控制挑剔如何更高质[控制欲是对生活考究的表征]
  • 天马行空的想象融入设计理念[形态表征]
  • 鼓足对新兴事物好奇心尝试创新[试着去做做出来]

创作精神在于做好自己

当我们想做好一件事，我们得学会专注但这不意味着单一，很多时候其实每个人都是没能做好自己的，这就告诫我们其实谈到说教，我们其实不够格，但又由于那可怜的表现欲，说到可怜这是因为没能做得更好真正意义上的近似完美，如果你深入了解史蒂芬·保罗·乔布斯，才够了解在表现这方面多么不足，完美主义要如何完美诠释。

当我们意识到做好自己时，我们才能足够专注，专注品质，创作总是不易，创作的目的和结果往往是不可知的，人们不知道真正需求直到把做好的摆出来，当我们苦心创作做出来，摆出来才能知道到底怎么，也许结果不尽人意，但过程算是最好奖励，我们收获经验与创作过程喜悦,我们做不到最好，但在成长，足够欣慰而不至于太过愚蠢

  • 本我、耐心、专注、超我

  • 试着去做总能做成

本篇日志就写到这里，里面多少还是存在问题的，笔者又懒得校验了，算是一次专注的尝试，日后再接再励，把学习、创作当成生活方式。
