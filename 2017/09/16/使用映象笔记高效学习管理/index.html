使用映象笔记高效学习管理

发表于 2017-09-16   |   分类于 技术栈   |     |   阅读次数

在对Head Frist html与css 通读学习过后，很惊讶这本神奇的书，通俗易懂全面而又细致的展开了相关知识点，于是在想web开发怎么能缺少Javascript，于是在safari books online上检索，发现就在这两个月出来《Head First Javascript programming》这本书，很幸运的在图灵社区找到了这本书的中译版前几节，在线学习体验还是很不错，但是只是读感觉效果不佳，所以试着使用映象笔记实践学习管理。关键字映象笔记Evernote

1.映象笔记能做什么

初步使用映象笔记，发现 Evernote 可以实时同步更新网页与桌面版，实现在线学习，离线线下两不误，同时协调微信通过关注我的映象笔记，官方介绍

1.一键保存网页
2.轻松圈出重点
3.快速捕捉精彩(印象笔记·剪藏，是保存网页的最佳方式，自动识别网页正文，智能去广告，轻松收藏给予你灵感的一切网页)
4.如何使用印象笔记·剪藏

2.映象笔记快捷键使用

a.打开chrome 拓展程序（插件），为Evernote 添加键盘快捷键，我这里Ctrl+1快捷打开Evernote 插件，使用Esc快速退出插件
b.chrome 拓展程序找到映象笔记打开选项,可以查看快捷键内容

2.1.几个常用快捷键

网页正文-A
屏幕截图-M
预览整页-Q（修改后）
PDF剪藏-p

3.Evernote灵活的标签和笔记本

观看一些大牛们使用技巧，发现了很多自己未能估计的问题，也是现如今更新博文的问题，标签的使用过于随意

3.1.笔记本管理

关于EN笔记本层叠结构大致包括笔记本组（貌似只能在桌面端创建）-笔记本-笔记，可以加数字开头便于排序

3.2.建立自己的标签使用原则

以实用为原则，混乱中建立秩序,来源：知乎

1.标签使用含糊

  • “小米”同时标注《我与王小米出游记》、《小米的绝妙吃法》和《小米手机简析》几条毫不相干的笔记

2. 标签使用含糊

  • “服装”标签可能标注《优衣库23条经营理念》，也可能标注《本年度秋季最流行搭配》
  • “电商”标签可能标注《马云谈如何踢足球》，也可能标注《马云谈淘宝战略》

3. 同义词标签

  • 同时出现 “Uniqlo 优衣库” “ 优衣库 Uniqlo”
  • 同时出现 “大锤” “王大锤”同时出现 “乔布斯” “Jobs”
3.4.1.基本属性纬度

之所以叫基本纬度是因为这一累标签具有“通用性”,在使用这一类标签的时候，我建议采取“固定格式”，为的是逻辑清晰和易于管理。这一大类标签分为几个小类：

1.地点：［@北京］
2.场景：［饭馆］［书店］［酒吧］
3.评分系统：［＃＃＃］［＃＃＃＃］［＃＃＃＃＃］
4.媒体：［book:逻辑学导论］［movie:冷山］［song:最後の一葉］
5.人物：［人名］或者［people：人名］
6.项目：［##项目名称］［##项目名称－子项目]
7.愿望与计划（GTD）： 事情[~to do］ 地点［~to go］ 物品［~to buy］
8.标记系统纬度：［＃＃需要回顾］［memo］

这里语义化TAG能够方便对于不断增加的笔记量来说，是快速定位查询内容的法门，合理使用笔记标签很重要

3.4.2.细分的内容纬度

1.为每个笔记本设计一套内容标签系统：比如我为我的【空间与家居 】这个笔记本就设计了[卧室、书房、客厅、厨房]这些标签
2.网络文章使用其自有的标签：现在很多网页都有自己的标签，可以作为参考
