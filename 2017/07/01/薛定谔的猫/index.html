薛定谔的猫

发表于 2017-07-01   |   分类于 科学   |   阅读次数

关于薛定谔的猫，是一个很有趣的实验模型，生活中总会碰到，介于两难的选择，可能无限美好也可能变得极其糟糕，而通常如果表现为不作为，两种情形臆想中就可能同时存在，而这就是薛定谔的猫的叠加态。

以下引自知乎（如何给没有高中数学基础的人解释「薛定谔的猫」？）

把一只猫放进一个封闭的盒子里，然后把这个盒子连接到一个包含一个放射性原子核和一个装有有毒气体的容器的实验装置。设想这个放射性原子核在一个小时内有50％的可能性发生衰变。如果发生衰变，它将会发射出一个粒子，而发射出的这个粒子将会触发这个实验装置，打开装有毒气的容器，从而杀死这只猫。根据量子力学，未进行观察时，这个原子核处于已衰变和未衰变的叠加态，但是，如果在一个小时后把盒子打开，实验者只能看到“衰变的原子核和死猫”或者“未衰变的原子核和活猫”两种情况。

个人认为这个实验有趣的几个点，第一放射性原子核发生衰变的随机性完全无法预测，第二自然认知所不待见的叠加态，第三严谨科学实验与猫联系，就显得并不那么严肃而变得有趣（当然个人认为实验是假想模拟运作）。

薛定谔的猫给我的启示感，宏观世界中未知的未来态必然是确定态，而实验确实是想证实这种认知的不完善性，自然而然联想到游离于两种态的并存态，只是试着打波自然认知的契机，而猫既死又活的悖论给我的感觉可能就在于尝试，消除不确定模糊态就是参与进去，也许可喜也许可悲。

从哲学上解释，而且还可以很”唯心“(该词非我本意，只是为了说明我在讨论什么)
薛定谔的猫是正统量子力学的经典描述，我把话题引申为佛学与量子力学是否可以互相描述。
佛典型的世界观：外境（色法）是不存在的，是由心变化而成，主张法界中一切（十八界）唯心造。
简单描述下量子力学的“唯心”道路
1、海森堡的测不准原理电子的动量p与位置q不能同时精确的确定，当确定精确的p时，q便无法确定；同样，当确定精确的q时，p便无法确定；或者p、q都十分模糊的确定。
2、电子/光子的波粒二象性这个大家都很熟悉，但可能没有人想过波粒二象性到底是什么意思，其实这是薛定谔猫的真实存在版，你无法确定光子是波动的能量还是实在的粒子，直到你去测量它。
我们可以简单的理解为量子世界中那个一直在困扰着科学家们的“观察者”就是佛学中的“心”，一切外物都是不存在/不确定的，直到你去观察/思考它。
无论是科学还是哲学，甚至是宗教学，这都是我们认识世界的手段而已，在人们将他们视为不共戴天的敌人的时候，也许他们已经握手言和。
菩提本非树, 明镜亦非台, 本来无一物, 何处染尘埃？
《上帝掷骰子么–量子物理史话》，这是一本关于量子物理的整个历史以及相关理论的简单介绍，好书。
作者：imajeason 来源：知乎

最后，抛开薛定谔的猫隐含的深层次的意义，就表面来看个人认为面对胶着态时倾向一方才能知道发展的好坏，也许薛定谔的猫能够成为母妻落水先救谁最好的答案，因为只有倾向一方后才能知晓事态结果。
